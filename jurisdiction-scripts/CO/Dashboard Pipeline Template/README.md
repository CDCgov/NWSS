### **Colorado Wastewater Surveillance Backend Pipeline Template**



This pipeline template was developed using **R version 4.3.1** and **RStudio version 2023.09.1 (Build 494)**. To minimize potential package compatibility issues, we recommend installing these or similar versions. Older versions may cause issues, particularly with the bsts library.



To get started, please refer to the “[Colorado Wastewater Surveillance Dashboard Template SOP](https://docs.google.com/document/d/1cvuIRo1dyiCyU9g5c-aWWQVtKbK9XdA_8kIx9pcBRdg/edit?tab=t.0)”, which provides detailed instructions for the setup process.
